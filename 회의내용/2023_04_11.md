먼저, cmds.ls 명령어를 사용하여 현재 씬에 있는 모든 오브젝트를 가져올 수 있습니다. 이 명령어는 long 플래그를 사용하여 모든 오브젝트의 전체 경로를 반환합니다.

다음으로, cmds.setAttr 명령어를 사용하여 각 오브젝트의 ID를 설정할 수 있습니다. setAttr 명령어는 오브젝트의 속성 값을 설정하는 데 사용됩니다. 이 명령어를 사용하여 objectId 속성을 설정할 수 있습니다.

아래는 예시 코드입니다.

python
Copy code
import maya.cmds as cmds

# 현재 씬에 있는 모든 오브젝트 가져오기
objects = cmds.ls(long=True)

# Object ID 초기값 설정
object_id = 1000

# 모든 오브젝트의 Object ID 설정
for obj in objects:
    cmds.setAttr(obj + '.objectId', object_id)
    object_id += 1
위 코드는 현재 씬에 있는 모든 오브젝트의 Object ID를 1000부터 시작하여 하나씩 증가시키면서 설정합니다. 이 코드를 실행하면 각 오브젝트의 objectId 속성이 설정됩니다.

다음은 이러한 작업을 수행하는 예제 코드입니다.

python
Copy code
import maya.cmds as cmds

# 현재 씬에 있는 모든 노드를 가져온다
all_nodes = cmds.ls()

# 각 노드의 참조 정보를 가져와서, 해당 참조가 어떤 에셋 파일에 로드되어 있는지 확인한다
for node in all_nodes:
    ref = cmds.referenceQuery(node, referenceNode=True)
    if ref:
        # 해당 레퍼런스가 로드된 에셋 파일 이름을 가져온다
        ref_file = cmds.referenceQuery(ref, filename=True)
        print("Node {0} is loaded from asset file {1}".format(node, ref_file))
    else:
        print("Node {0} is not a reference".format(node))
이 코드는 현재 씬에 있는 모든 노드를 가져와서, 각 노드가 레퍼런스인지 확인합니다. 레퍼런스라면, 해당 레퍼런스가 로드된 에셋 파일 이름을 가져와서 출력합니다. 레퍼런스가 아니라면, 해당 노드가 레퍼런스가 아님을 출력합니다.

이 코드를 실행하면, 현재 씬에 있는 모든 노드 중에서 레퍼런스로 로드된 노드들이 어떤 에셋 파일에서 로드되었는지 확인할 수 있습니다.
